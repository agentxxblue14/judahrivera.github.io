# Molongski Proposal Generator

AI-powered freelancer productivity tool using the Molongski Method. Paste a job post, get a KYN score and 3 KISS proposal variants.

## Setup

```bash
cd molongski-app
cp .env .env.local  # Add your real ANTHROPIC_API_KEY
npm run install:all
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:3001

## Tech Stack

- React + Vite (frontend)
- Node.js + Express (backend)
- Anthropic Claude API (AI)
